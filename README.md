# wanhsiyuan-site

Static site for [wanhsiyuan.com](https://wanhsiyuan.com). Astro + Cloudflare Pages.

## Develop

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

Outputs to `dist/`.

## Curve refresh

`public/curves/sleeves.svg` and `src/data/sleeves.json` are generated by
`~/code/scripts/export_public_curves.py` and committed to this repo by a
launchd job on the trading box. Do not edit by hand.
